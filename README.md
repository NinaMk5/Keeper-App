# Keeper App

A minimal React note-taking example built with Vite and Tailwind.

## Features

- Create notes with a title and content
- Delete notes

## Project structure

- `src/` — application source
	- `App.jsx` — top-level app and state
	- `main.jsx` — app entry
	- `components/` — presentational components (`Header`, `CreateArea`, `Note`, `Footer`)
	- `index.css`, `App.css` — basic styles

## Requirements

- Node.js 18+ (or compatible)
- npm or yarn

## Setup

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

3. Open http://localhost:5173

## Scripts

- `npm run dev` — start Vite dev server
- `npm run build` — build production bundle
- `npm run preview` — preview production build locally
- `npm run lint` — run ESLint
